# 115 离线下载插件

> 这是一个快速将磁力链接离线到 115 网盘的插件, 您安心飙车, 磁力链交给我

### 功能

1. 识别磁链离线
2. 识别未带火车头的磁力链接
3. 欣赏妹子
4. aria2 导出下载

### 场景

场景简介：

- 场景一：【一键离线】 识别页面的所有磁力链，并在磁力链左边加一个小图标，点击即可离线
- 场景二：【文本离线】不带 magent 头部的文本，使用鼠标右键发送文本到 115，即可离线
- 场景三：【aria2下载】使用aria2下载到挂载硬盘，nas等远程设备
- 场景四：老司机们自己挖掘咯
