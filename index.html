Capabilities

少点复制粘贴，多点自动化。

围绕「发现链接 → 提交离线 → 导出下载」这条路径重做信息架构，核心能力一眼可见。

A

磁链 / Hash 识别

自动识别页面里的 magnet 链接、未带协议头的磁力文本，以及 SHA1 文本，减少手动整理。

B

页面一键离线

在识别到的链接旁加入快捷入口，点击即可把任务发送到 115 离线下载流程。

C

右键文本离线

选中网页中的 Hash 或磁力文本，使用浏览器右键菜单直接提交，不必新开页面。

D

aria2 导出

把 115 或夸克网盘任务导出给 aria2，适合 NAS、挂载硬盘和远程下载机。

Download

选择稳定的安装方式。

优先使用 Chrome 商店安装；如果商店不可用，可以从 GitHub Releases 获取压缩包并手动加载。新版 Chrome 通常建议加载解压后的扩展目录，而不是直接拖入 CRX。

Install

手动安装，只需要三步。

如果你下载的是 ZIP 包，请按浏览器扩展的开发者模式流程加载。

下载并解压

从 GitHub Releases 或本地 ZIP 包下载插件，解压到一个固定目录，后续不要随意删除。

打开扩展页面

在地址栏输入 chrome://extensions/，打开右上角「开发者模式」。

加载扩展目录

点击「加载已解压的扩展程序」，选择刚才解压出来的文件夹，完成安装。

FAQ

常见问题。

把 README 中的关键排障信息前置，用户不用翻文档也能先解决高频问题。

明明付费了，却一直显示非会员？

先检查系统时间是否准确，时间偏差可能影响会员状态识别与接口校验。

115 页面没有导出按钮？

确认插件设置里已经开启 aria2 开关，然后尝试点击 115 页面里的「文件首页」刷新入口。

浏览器通知没有出现？

检查系统勿扰模式、浏览器通知权限，以及当前浏览器是否被系统允许发送通知。

本地播放器无法播放流？

在播放器里配置对应 user-agent，再重新尝试播放下载流。
